# M5Stack_ambient_sensor_unit_test
M5Stack の環境センサユニットを [Seeed Studio XIAO](https://www.seeedstudio.com/xiao-series-page) で使用するための動作確認<br>
ライブラリ付属のサンプルコードをもとに作成<br>

## 確認済みセンサ
* M5Stack DLight Unit
* M5Stack ENV Ⅲ Unit
* M5Stack ENV Ⅳ Unit
* M5Stack ENV Pro Unit
* M5Stack TMOS PIR Unit
